# Software used in this exercise

This exercise uses a number of pieces of scientific software that are commonly used, but must be installed separately. Most of these (maybe all?) have [Bioconda](https://bioconda.github.io/) packages, so that may be the easiest way to install them.

Software used in this exercise includes:

- bwa-mem2 - <https://github.com/bwa-mem2/bwa-mem2>
- seqkit - <https://bioinf.shenwei.me/seqkit/>
- SRA toolkit - <https://github.com/ncbi/sra-tools/wiki/01.-Downloading-SRA-Toolkit>
- BLAST - <https://www.ncbi.nlm.nih.gov/books/NBK569861/>
- samtools - <https://www.htslib.org/>
- SAUTE - <https://github.com/ncbi/SKESA>
- AMRFinderPlus - <https://github.com/ncbi/amr/wiki/Installing-AMRFinder>
